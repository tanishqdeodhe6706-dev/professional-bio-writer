# Professional Bio Writer

## 📌 Description
This project generates professional bios based on user input such as name, role, and skills.

## 🚀 Features
- Generates 100, 150, and 200 word bios
- Saves output to file
- Simple and user-friendly

## 🛠️ Tech Used
- Python

## ▶️ How to Run
1. Open terminal
2. Run:
   python app.py
3. Enter details

## 📂 Output
Generated bios are saved in the outputs folder.
